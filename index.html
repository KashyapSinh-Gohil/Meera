Quick Start

Installation

To get started with MeeraCV, you need to install the library. You can do this using pip, the Python package manager. Open your terminal or command prompt and run the following command:

pip install meeracv

Basic Usage

Once you have installed MeeraCV, you can start using it in your Python scripts. Below is a simple example demonstrating how to read an image, convert it to grayscale, apply Gaussian blur, and save the result:


import meeracv as mcv

# Read an image from file
img = mcv.imread('image.jpg')

# Convert the image to grayscale
gray = mcv.cvtColor(img, mcv.COLOR_BGR2GRAY)

# Apply Gaussian blur to the grayscale image
blurred = mcv.GaussianBlur(gray, kernel_size=(5,5), sigma=1.0)

# Save the processed image to a new file
mcv.imwrite('result.jpg', blurred)

Advanced Features

MeeraCV also provides advanced features such as face detection and gesture recognition. Below are examples of how to use these features:

Face Detection

To detect faces in an image, you can use the FaceDetector class. Here’s how to do it:


from meeracv import FaceDetector

# Initialize the face detector
detector = FaceDetector()

# Detect faces in the image
faces = detector.detect_faces(img)

# Print the detected face coordinates
print("Detected faces:", faces)

Gesture Recognition

For recognizing hand gestures, you can use the HandGestureRecognizer class. Here’s an example:


from meeracv import HandGestureRecognizer

# Initialize the hand gesture recognizer
gesture_recognizer = HandGestureRecognizer()

# Get hand landmarks from the image
landmarks = gesture_recognizer.get_hand_landmarks(img)

# Recognize the gesture based on the landmarks
if landmarks:
    gesture = gesture_recognizer.recognize_gesture(landmarks[0])
    print("Recognized gesture:", gesture)
else:
    print("No hands detected.")
